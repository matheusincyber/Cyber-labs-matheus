# Contributing

Thanks for helping improve this learning repo! A few simple rules:

- Be kind and constructive.
- Keep sensitive data out of commits (no private keys, secrets, lab flags unless redacted).
- Use branches for non-trivial changes: `feature/<short-desc>`.
- Open a Pull Request and add a short description of what changed.
- If adding new labs, include a `labs/<lab>-template.md` that follows existing structure.

Small changes (typos, formatting) may be merged directly.
