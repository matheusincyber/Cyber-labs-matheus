## How to use (quick start)
1. Download the ZIP from this page and extract it locally.  
2. Read `README.md` then open `docs/week01-networks.md` to start.  
3. Labs: open `labs/mr-robot-ctf.md` and `labs/wireshark-incident-response.md`.  
4. Scripts: run locally in a safe environment (VM) — see `scripts/README.txt`.

**Important**: never run offensive tools against systems you don't own or have permission to test.
