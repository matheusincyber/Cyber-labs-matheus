# Useful Websites (study & reference) - short notes
- GTFOBins — https://gtfobins.github.io/ — catalog of ways to abuse common binaries for privilege escalation (reference only).
- VirusTotal — https://www.virustotal.com/ — upload hashes/files for multi-engine analysis.
- Hashes.org / CrackStation — hash lookup/rainbow tables.
- OWASP — https://owasp.org/ — web app security and top 10.
- TryHackMe — https://tryhackme.com/ — lab rooms and learning paths.
- SecLists (GitHub) — https://github.com/danielmiessler/SecLists — wordlists for enumeration.
- John the Ripper — https://www.openwall.com/john/ — password cracking tool.
- Nmap NSE documentation — https://nmap.org/book/nse.html
(use links responsibly; these are for study in authorized labs)
